主題介紹

The True Hackers

今年的活動主題「The True Hackers」，將聚焦於真正的駭客精神。駭客發源於80年代的 MIT 人工智慧實驗室，其對抗壟斷、反抗權威、推崇自由開放、重視技術實作的精神至今 在世界發揚光大。初代駭客們更相信資訊不應被壟斷，致力於將電腦普及， 他們相信電腦可以改善人的生活，電腦不應被鎖在企業與政府的層層權威與官僚當中。

為了避免資訊科技落入政府與大企業的壟斷之中，駭客們提倡分散式與開放的系統， 他們更預見資訊科技的全面普及，雖然可以為人類帶來眾多便利，但若資訊科技的力量 被政府和企業壟斷掌控，便會形成一個各方面都被監控的社會。他們的洞見在史諾登揭露 NSA 全球全面性無差別的竊聽計劃之後被證明不是危言聳聽。

現今駭客們更加大聲地疾呼隱私權的保障，以捍衛最基本的人權—自由。除此之外， 他們的理念也影響了諸如自由軟體、開放政府、資訊安全與自造者運動等多種領域。 這次的年會除了延續之前的理念推廣外，也希望讓參與的學生與民眾了解駭客精神的 真諦思想，與駭客精神在各種不同領域的展現。

議程內容

克莉絲汀 - Project Squirrel臺大開發者社群

身為學生的你，是否曾在校園生活中感到不便卻無力改變？有沒有曾經幻想過，有一套 「學生秘書系統」能將選課程式、社團訊息、美食地圖等功能集於一身？讓我們跟著 NTUIP （臺大學生資訊平台）發起人的腳步一起來見證，看這些大學生如何透過雙手、改變成真。

RSChiang - 開源社的第一年

對於沒有接觸過 Open source 的人們來說，「開源」這個詞既不著邊際又容易被誤解。 我們邀請到臺大開源社的創始社長，向大家分享在校園推廣資訊透明的心路歷程。 聽聽一群懷抱熱情的學生，如何用「聽得懂」的語言推廣開源，又是如何透過舉辦講座、 開放課程、漏洞提報，甚至在研發專案之餘主動關心校園議題，貫徹自由開放的價值。

Dylandy - 帶你的朋友一起寫程式吧

在這個資訊科技做為科技領頭羊的時代，有很多人也都想要學習寫程式，很多朋友不管是有需要、 被逼的還是自發性想學習寫程式，卻往往不得其門而入；市面上的教科書參考書五花八門， 寫得也都洋洋灑灑，在初入門的人眼中只會眼花撩亂，而且習慣於人類的思想，無法理解電腦的 語言，也讓有心來學的人多半打退堂鼓。到底要怎麼幫助他們才會比較好呢？Dylandy 這個 talk ，就要告訴你如何才能讓新手可以開開心心的進入資訊領域往上爬。

Arbiter - 初探機器學習: 教電腦如何看手勢

十九世紀晚期，打字機問世；二十世紀中葉，打字機跟計算機結合運用，鍵盤這個名詞因而誕生； 二十一世紀初，觸控面板隨處可見；為了有更好的使用者經驗，技術隨著時代的推演而更新， 而現在 Arbiter 將帶領我們使用不同的使用者介面控制方式，讓電腦了解你的手勢，以動感的方式 控制你的電腦，相信這會是一個全新體驗。

浪打(0x1A3BDA) - 電吉他效果器，基於ARM微控制器

每一個學電子樂器並且對音樂有狂熱的人都會有一台效果器，效果器不只可以幫忙修飾電子樂器的 音色，還可以改變其音訊，而這使音樂人們可以透過改變其設定或是透過不同效果器的組合，營造 出音樂的無限可能性；讓我們來看看浪打如何從 ARM 處理器架構去調控效果器的控制系統，進而 譜出動人的生命樂章。

colin - 可程式化組合邏輯控制方塊

「寓教於樂」是兒童教育的一大理想，也是各界不斷致力改進的議題。然而，就兒童的資訊教育而言 ，該如何讓小朋友藉由玩樂，產生對資訊的興趣，進而能夠深入鑽研？且看 colin 以孩童都會玩的 積木作為開發基礎，結合電子程式與各種硬體週邊，讓使用者在玩樂中習得基本的程式邏輯與軟硬體 概念，甚至使用各種週邊組合出有具體功能的裝置。

Jennifer - Python製作！用腳踏車控制的Android遊戲

想想你曾經騎過的腳踏車，再想想 Android 手機裡面的遊戲。看似八竿子打不著，但 Jennifer 透過 Python 、現在當紅的 Raspberry Pi，與安裝在腳踏車上的感應器，就可以讓腳踏車搖身一 變，成為 Android 遊戲 APP 的控制器，讓遊戲體驗更為逼真，融入真實世界。遊戲角色的移動與 行為，全憑你的「騎」技！

贊助單位

晶睿通訊 VIVOTEK

晶睿通訊 (http://www.vivotek.com) 是國內第一大監控方案 提供廠商，公司團隊集合了國內視訊、語音及通訊產品的研發與行銷的專才，掌握影音壓縮核心技術 ，深耕網路通訊協定、機電整合，多媒體系統單晶片及軟韌體研發。近年更致力於軟硬體整合，打造 一流的軟體開發環境。希望憑藉台灣硬體優勢，拉抬台灣軟體能見度。軟體為王的時代來臨，我們深信 唯有軟硬結合才是未來的趨勢，歡迎到我們的網誌 http://vvtk-digest.blogspot.tw/ 逛逛。
